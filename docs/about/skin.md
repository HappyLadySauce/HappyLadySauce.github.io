# HppleCraft

HappleCraft是我搭建的一个Minecraft的服务器，无盈利纯公益服务器。

服务器使用自建皮肤站(www.happlelaoganma.cn)认证，没有正版也能使用皮肤游玩。

相关的QQ群: 940648979

在这里我会放出最新周目的情况和说明:

## 第二周目: 植物魔法 + 方块宝可梦的轻量MOD生存

### 基本信息

MC版本: 1.20.1

服务器端: forge + plugins

支持人数: 24人

### 主要模组

植物魔法(无附属)和方块宝可梦，辅助MOD有FTB全家桶，精妙背包，车万女仆和农夫乐事(有附属)等
还有一些优化MOD外加铷和光影。

### 其他说明

本周目在12月之前都将稳定运行，大家放心游玩。只不过最近一个周在测试中，可能会增加或者减少模组。
